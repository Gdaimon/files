Restful Services con Play Framework 2.x, MySQL y Seguridad con JWT

Descripción

Play es un framework que en su versión 2.x ha madurado mucho. Permite hacer desarrollo Web incluyendo WS Restful, puedes utilizar Java o Scala y podemos agregar seguridad utilizando JWT. Descubre como crear un WS desde 0 en esta charla.
